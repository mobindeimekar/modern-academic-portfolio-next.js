# Modern Academic Portfolio – Next.js

A modern academic portfolio website built with Next.js (App Router), Tailwind CSS, and Framer Motion.

## 🚀 Features

- ⚡ Performance-optimized
- 🎨 Smooth stagger animations
- 📱 Fully responsive
- 🔍 SEO-friendly structure
- 🧩 Clean and scalable architecture

## 🛠 Tech Stack

- Next.js 14+
- Tailwind CSS
- Framer Motion
- Swiper.js

## 📦 Installation
```bash
npm install
npm run dev
